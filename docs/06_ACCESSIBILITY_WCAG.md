# Accessibility Baseline (WCAG 2.2 AA)

## Non-negotiables
- Navegacion completa por teclado.
- Focus visible y orden de tabulacion coherente.
- Contraste minimo AA para texto y controles.
- Etiquetas accesibles para todos los controles.
- Estados y mensajes con soporte para lectores de pantalla.

## Interaction Patterns
- Decisiones narrativas activables con teclado y tactil.
- Tiempos de votacion configurables y pausables.
- No depender solo de color para feedback.

## QA Checklist
- Test manual con teclado en todas las vistas clave.
- Reflow correcto en movil sin perdida funcional.
- Validacion con herramientas automaticas + revision humana.
